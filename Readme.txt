Бот написан на языке Python 3.10. Для работы бота необходимо установить библиотеки из файла requirements.txt.
Бот использует API с сайта rapidapi.com.
Необходимо создать файл .env по образцу файла .env.template, в котором будут храниться RAPIDAPI_KEY и токен бота.
В проекте используется база данных sqlite3 и ORM peewee, а также ведется логирования с помощью библиотеки loguru.

Бот состоит из модулей:

    main - основной модуль для запуска бота.
    data_base - модуль с описанием моделей базы данных
    handlers - директория с командами бота
    api_requests - директория с запросами к API
    keyboards - клавиатура бота
    config - настройки конфигурации бота
    states - состояния бота
